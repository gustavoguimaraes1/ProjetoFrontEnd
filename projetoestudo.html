Sobre

O estudo de Front-End foi iniciado com o objetivo de desenvolver habilidades em criação de interfaces modernas, acessíveis e responsivas.

O foco principal é unir design, programação e experiência do usuário para criar projetos funcionais e visualmente organizados.

Este projeto também representa minha preparação para atuar futuramente em equipes de desenvolvimento e inovação tecnológica.

Através do HTML, CSS e JavaScript, estou construindo uma base sólida para evoluir futuramente para tecnologias modernas como React e APIs.
